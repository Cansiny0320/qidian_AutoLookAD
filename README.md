# 基于[autoxjs](http://doc.autoxjs.com/)起点福利中心自动脚本

每天起点几毛钱的抵扣卡白嫖,自己看总觉得很亏,就几毛钱,食之无味,弃之可惜

搜了一下居然没有自动脚本, 自己基于autojs 做了一个,打包为app

缺点:

每次都要重新开无障碍, 这个很麻烦,除非把他扔到自启动然后不关闭,定时执行.不知道有没有人有更好的办法
